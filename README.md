#Responsive Landing Page

A modern, responsive landing page with an interactive navigation menu and smooth scrolling functionality. The website features a clean design with distinct sections for Home, About, Services, and Contact.

## Features

- Fully responsive design that works on all devices
- Interactive navigation menu with smooth scrolling
- Color-coded sections (Blue for Home, Orange for About, Green for Services, Blue for Contact)
- Mobile-friendly hamburger menu
- Smooth transitions and animations
- Clean and modern UI design

## Project Structure

```
responsive-landing-page/
├── index.html      # Main HTML file with all content sections
├── styles.css      # CSS styles including section colors and animations
└── script.js       # JavaScript for smooth scrolling and navigation
```

## Usage

- The navigation menu is fixed at the top of the page
- Click on any menu item to smoothly scroll to that section
- The active section is highlighted in the navigation menu
- On mobile devices, use the hamburger menu to navigate

## Sections

1. **Home Section**
   - Welcome message in blue color
   - Hero section with background image

2. **About Section**
   - Orange background
   - Personal introduction

3. **Services Section**
   - Green background
   - Services information

4. **Contact Section**
   - Blue background
   - Contact information including phone numbers and email

## Customization

1. Colors:
   - Change section colors in `styles.css`
   - Modify navbar colors in `styles.css`

2. Content:
   - Update text content in `index.html`
   - Modify section layouts in `styles.css`

3. Animations:
   - Adjust transition timings in `styles.css`
   - Modify scroll behavior in `script.js`
